    Spotify Explorer

    Made by luungoc2005

    An artist explorer app for Spotify, making use of Spotify Web API for search, suggestions and track samples.

    Region is set to US by default, since Spotify is not available where I live.

    Libraries usage

    Spotify Web API v1 including search, find related artists and find related tracks

    jQuery 3 for various UI and Ajax functions

    Bootstrap 3 for theming and glyph icons
